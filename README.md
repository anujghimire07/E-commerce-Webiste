# E-commerce Website

A modern and responsive **React + Vite** e-commerce website built as a learning project over 15 days, combining core React concepts into a full-fledged project. Users can browse products, add items to the cart, and enjoy a smooth shopping experience.

🔗 **Live Demo:** [E-commerce Website](https://e-commerce-webiste-three.vercel.app/)

---

## Features

- Browse products by category
- Add, remove, and update items in the shopping cart
- Responsive design for desktop and mobile
- Smooth animations and interactive UI
- Built using modern React concepts learned step by step

---

## Learning Progression

The project was built following a structured **15-day React learning path**:

| Day | Topic | Concepts Learned |
|-----|-------|-----------------|
| 1 | useState basics | State creation, updating state, re-rendering, event handling (onClick), build counter |
| 2 | useState advanced | Arrays & objects in state, immutability, updating lists, build dynamic list |
| 3 | Event handling deep dive | onChange, onSubmit, controlled inputs, build form |
| 4 | Forms advanced | Multiple inputs, validation, error messages, build signup form |
| 5 | useEffect basics | Side effects, dependency array, fetch API |
| 6 | useEffect advanced | Cleanup, avoiding infinite loops, build timer |
| 7 | Lists & keys | map(), unique keys, rendering lists, build todo app |
| 8 | Conditional rendering | Ternary, &&, UI toggles |
| 9 | React Router basics | Routes, links, navigation |
| 10 | Router advanced | Params, nested routes |
| 11 | useContext | Global state, avoid prop drilling |
| 12 | Custom hooks | Reusable logic (useFetch) |
| 13 | useRef | DOM access, form focus, avoiding re-render |
| 14 | Performance | useCallback, useMemo basics |
| 15 | React project | Combine all concepts into a full e-commerce app |

---

## Technologies Used

- **Frontend:** React (with Hooks)
- **Build Tool:** Vite
- **Routing:** React Router
- **State Management:** useState, useContext, custom hooks
- **Styling:** CSS / SCSS
- **Deployment:** Vercel
- **APIs:** Optional fetch API for products
